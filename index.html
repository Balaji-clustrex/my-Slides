Web3 JS Tutorial

Introduction

  1. Ethereum works in app using the web3 object provided by the web3.js library
  2. It communicates to a local node through RPC calls
  3. Works with any Ethereum node
  4. Web3 Can be integrated into the project in following ways
    • Npm: npm install web3
    • Bower: bower install web3
    • Meteor: meteor add ethereum:web3
    • Vanilla: link the dist./web3.min.js
